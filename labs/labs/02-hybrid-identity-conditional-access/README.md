# Hybrid Identity & Conditional Access Lab

## Objective
Simulate a hybrid identity environment and implement authentication controls using Microsoft Entra ID, focusing on MFA enforcement, Conditional Access behavior, and identity-based access decisions.

## Environment
- Microsoft Entra ID (Azure AD)
- Windows Server Active Directory (conceptual hybrid identity)
- Test user accounts

## What This Lab Covers
- Cloud-based identity management
- User and group creation in Entra ID
- Multi-Factor Authentication (MFA)
- Conditional Access policy concepts
- Authentication success and failure scenarios

## Key IAM Activities
- Created cloud users and groups in Entra ID
- Enabled and enforced MFA for user accounts
- Implemented Conditional Access policies based on user and access conditions
- Tested authentication behavior before and after policy enforcement
- Simulated failed login attempts and access denials

## Validation
- Verified MFA prompts during authentication
- Confirmed Conditional Access policy enforcement
- Validated successful and blocked login attempts
- Reviewed sign-in activity for authentication events
