# Enterprise IAM Architecture & JML Automation Lab

## Objective
Design and implement an enterprise-style Identity and Access Management (IAM) architecture focused on user lifecycle management, RBAC, automation, and least-privilege access.

## Environment
- Windows Server Active Directory
- Microsoft Entra ID (Azure AD)
- PowerShell automation

## What This Lab Covers
- Organizational Units (Users, Admins, Service Accounts)
- User and group management
- Role-Based Access Control (RBAC)
- Joiner / Mover / Leaver (JML) lifecycle workflows
- PowerShell automation for identity tasks
- Least-privilege access enforcement

## Key IAM Activities
- Provisioned users and assigned role-based group access
- Modified user access during role changes
- Deprovisioned users and validated access removal
- Reviewed group memberships to identify excessive privileges

## Validation
- Verified authentication and authorization behavior
- Tested access before and after role changes
- Confirmed access removal after deprovisioning
