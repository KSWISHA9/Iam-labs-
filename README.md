# Identity & Access Management (IAM) Labs

This repository contains enterprise-style Identity and Access Management (IAM) labs designed to demonstrate hands-on experience with identity lifecycle management, access controls, authentication, governance, and incident response.

These labs mirror real-world IAM workflows performed by IAM Analysts, Access Management Analysts, and Junior Identity Engineers.

---

## Core IAM Skills Demonstrated

### Identity & Directory Services
- Active Directory (users, groups, OUs)
- Microsoft Entra ID (Azure AD)
- Hybrid identity concepts
- User lifecycle management (Joiner / Mover / Leaver)

### Access Management
- Role-Based Access Control (RBAC)
- Group-based access enforcement
- Least-privilege access models
- Permission reviews and remediation

### Authentication & Authorization
- Multi-Factor Authentication (MFA)
- Conditional Access concepts
- Authentication troubleshooting
- Account lockout analysis

### Identity Governance & PAM
- Identity Governance concepts (IGA)
- Access certifications and reviews
- Privileged account separation
- Just-In-Time (JIT) access concepts
- Privileged access logging and review

### Automation & Troubleshooting
- PowerShell automation for identity lifecycle tasks
- Bulk user provisioning and deprovisioning
- Identity-related incident response
- Root cause analysis and validation

### Security & Compliance Awareness
- Least privilege enforcement
- Identity-related audit evidence
- Log review for authentication events
- Incident documentation and remediation

---

## Labs Included

### 1. Enterprise IAM Architecture & JML Automation
Designed and implemented an enterprise IAM architecture using Active Directory and Microsoft Entra ID, focusing on user lifecycle automation, RBAC, and least-privilege access.

### 2. Hybrid Identity & Conditional Access
Simulated hybrid identity environments and implemented Conditional Access and MFA policies to control authentication behavior.

### 3. Identity Governance & Privileged Access Management
Implemented identity governance workflows, access certifications, and privileged access controls aligned with enterprise IAM platforms.

### 4. Identity Incident Response
Investigated and resolved identity-related incidents including account lockouts, authentication failures, and excessive privileges.
